## Glaze

Raycast's newly released desktop app theme customization tool, Glaze, is quite impressive. Arguably more valuable than LobsterAI — scaled up, it could rival the App Store, and an Apple acquisition isn't out of the question. The future where everyone builds and publishes apps by just describing them is getting real. Most people don't have beta access yet, but the new Glaze wallpaper series is available for download now.

https://www.glazeapp.com/

## Agents UI

LiveKit released Agents UI, a voice interaction UI framework — similar to what ElevenLabs open-sourced recently. The shader animation on the landing page caught my eye — it appeared to be Unicorn Studio's Aura effect layered with custom code. So I had Claude Code analyze the source, and it replicated the effect in no time. Then I extended it with custom theme colors and different textures.

This would have been unthinkable before. As a designer, even with prior coding experience, pulling off complex animations seen online was a painful, time-consuming learning curve. Now, any visual effect I encounter can be replicated — most within minutes. With AI, I feel genuinely unstoppable.

https://livekit.io/ui

## Resources

### ASC11

An online ASCII art generator that converts images and video into multiple ASCII styles — classic characters, Braille, halftone, particles, and more. Features a 10-band equalizer-style parameter panel, multiple dithering algorithms, color modes, and FX presets, with export to Canvas or WebGL.

https://asc11.com/

### ASCII Clouds

An interactive ASCII cloud generator that renders real-time cloud animations composed of ASCII characters using noise algorithms. Adjustable cell size, amplitude, speed, hue, contrast, with built-in presets like terminal, retro CRT, cosmic, and fog.

https://caidan.dev/portfolio/ascii_clouds/

### The Gridcn

A Tron: Legacy-inspired shadcn/ui theme component library featuring 6 neon color schemes named after Greek myths, 50+ components, HUD interface elements, and Three.js 3D effects. Built with React + Tailwind + TypeScript, fully free and open-source.

https://thegridcn.com/
