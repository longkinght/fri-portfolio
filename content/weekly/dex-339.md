HeroUI shipped version 3.0 last week. This is arguably the UI component library with the best design style and attention to detail available right now. They also have a HeroUI Pro tier, currently half price.

## Resources

### Flow Icons

A premium VSCode icon pack with hundreds of file type and folder icons. Supports multiple folder color schemes (blue, brown, gray, green, and seven more), customizable arrows and folder icon display, with a consistent and highly recognizable style. Free base version; the complete pack is $5 with future updates included. Note: the original doesn't support Zed — a community fork fixes that.

https://flow-icons.pages.dev/

### Radiant

An open-source shader and effects library offering 50+ ready-to-use web shaders — zero dependencies, MIT licensed. Each shader is a self-contained single-file HTML, ready to copy-paste into Claude Code, Cursor, or other AI coding tools for customization. Inspiration sources range from The Weeknd and Beyonce to Dua Lipa.

https://radiant-shaders.com/

### ASCII Magic

A feature-rich online ASCII art generator that converts images and video (including GIFs and MP4s) into character art. Offers brightness, edge detection, and dot-matrix rendering modes with adjustable character sets, coverage, and density. Advanced features include mask painting, background blur, color blend modes, and animated export.

https://www.ascii-magic.com/

### Studio

A generative design tool collection from neato.fun, containing eight modules: ASCII, Blocks, Dither, Gradients, Lines, Organic, Plotter, and Topo. Each module is an independent parametric visual generator for quickly creating abstract textures, patterns, and visual assets.

https://studio.neato.fun/

### More Resources

- **Motion Core** - A Svelte-native motion component library integrating GSAP and Three.js, with CLI for quick initialization and component scaffolding. Built for Svelte projects needing high-quality motion design, 3D canvases, and interactive animation. https://motion-core.dev/
- **Fonttrio** - A font pairing tool for shadcn/ui projects, offering 49 curated three-font combinations (heading, body, monospace) with complete typographic scales and CSS variables, installed via npx. https://www.fonttrio.xyz/
- **Loftlyy** - A brand identity exploration platform with a growing database of real brands. Browse logos, color schemes, and design systems for inspiration and reference. https://www.loftlyy.com/en
- **Motionwind** - A build-time animation library that brings Framer Motion into Tailwind CSS class names. Uses a Babel plugin to compile animate-* classes into motion components, supporting gestures, scroll triggers, spring physics, drag, and the full Motion API — zero runtime overhead. https://www.motionwind.xyz/docs
- **FragCoord.xyz** - A shader effect exploration and sharing platform hosting 2D/3D shader works for online preview and learning GLSL implementation techniques. http://FragCoord.xyz
- **OKLCH Gradient Generator** - A gradient generator based on the OKLCH perceptual color space. Adjust lightness, chroma, and hue for two color stops, export as Tailwind CSS or download as image. Produces more perceptually uniform gradients than traditional RGB. https://oklch-gradient.vercel.app/
- **Color_Therapy_001** - Limited edition digital wallpapers by Rik Oostenbroek — 6 high-resolution phone wallpapers exploring color and soft transitions. https://store.rikoostenbroek.com/products/digital-products-example-product-3

## Cases

### Ironhill

An Australian creative design studio site built around the narrative of "a place you won't find on any map." Scroll-driven exploration tells a mythology-infused brand story, weaving whiskey, beer, and drinking water products into a fantasy world of forest guardians. The visual style is distinctive, the copy is poetic, and it's an outstanding example of brand storytelling meets immersive web experience.

https://ironhill.au/

### John Hanacek

A charming personal portfolio. Led the redesign of Nanome's VR molecular design platform. The site showcases diverse projects spanning XR interfaces to agentic systems, and offers Claude Code coding tutoring — endorsed by Kevin Kelly and others.

https://www.johnhanacek.com/

### Mesh

Clay, the AI-powered personal CRM, rebranded to Mesh with a fresh identity. The product itself remains largely unchanged.

https://me.sh/
