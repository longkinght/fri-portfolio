After heavy use of various design-related skills recently, a clear pattern emerged: there are too many, most overlap significantly, and running all of them on a single project is wasteful — it burns tokens and creates conflicts. Here's a roundup of the design skills landscape right now:

---

## Design Skills Roundup

### Impeccable

An enhanced frontend-design skill with 17 design commands (/polish, /audit, /distill, etc.).

https://impeccable.style/

### UI Skills

Contains 15 independent skills: baseline-ui (Tailwind design consistency), fixing-accessibility (accessibility audits), fixing-metadata (SEO metadata), fixing-motion-performance (animation performance), 12-principles-of-animation (Disney animation principles), and more.

https://www.ui-skills.com/

### Taste Skill

Includes taste-skill (frontend design), redesign-skill (upgrading existing projects), output-skill (forcing AI to write complete code), and soft-skill (making things look and feel expensive — covering premium typography, generous whitespace, layered card layouts, fluid spring animations, and floating navigation).

https://github.com/Leonxlnx/taste-skill

### Better Icons

MCP server + skill for searching and fetching 200K+ icons, with automatic learning of commonly used icon libraries and direct sync to your project's icons directory.

https://github.com/better-auth/better-icons

### UI Design Brain

A best-practices library covering 60+ components, including layout patterns, accessibility rules, and anti-patterns, supporting 5 design styles (SaaS, minimal, enterprise, creative, data dashboard).

https://github.com/carmahhawwari/ui-design-brain

### AI Kit by Motion (Paid)

Motion+ members only. Includes an animation expert skill, performance auditing, CSS spring generation, transition visualization, and documentation search tools.

https://motion.dev/docs/ai-kit

### UI UX Pro Max

Offers 50+ design styles, 97 color schemes, and 9 tech stacks for comprehensive UI/UX design intelligence.

https://github.com/nextlevelbuilder/ui-ux-pro-max-skill

### Anthropic frontend-design

Anthropic's official skills repository with examples spanning creative design, technical development, enterprise communication, and document processing skills (docx, pdf, pptx, xlsx).

https://github.com/anthropics/skills

### Vercel Web Design Guidelines

Vercel's official skills collection — the web-design-guidelines skill alone contains 100+ UI audit rules.

https://github.com/vercel-labs/agent-skills

### Designer Skills Collection

63 skills and 27 commands across 8 plugins: user research, design systems, UX strategy, UI design, interaction design, prototype testing, design ops, and design toolkit.

https://github.com/Owl-Listener/designer-skills

### Emil Skills ($99)

Animation skills from animation.dev, sold separately.

https://animations.dev/learn/emil-skill

### Skills by InterfaceCraft ($199)

Requires purchasing the course to access.

https://www.interfacecraft.dev/

## Resources

### macOSicons

A veteran site housing 30,000+ free macOS-style icons contributed by the design community. Mentioning it again because of the recent redesign — in case anyone missed it.

https://macosicons.com/

### Grainrad

A browser-based creative media processing tool that transforms images, video, GIFs, and even 3D (GLB) files in real time into 15+ stylized effects — ASCII, dither, halftone, edge detection, pixel sorting, VHS, and more. Comes with a preset system and exports to PNG, SVG, MP4, and other formats.

https://grainrad.com/

### OpenUI

An open-source standard toolkit for generative UI. Developers register their component libraries with defineComponent, and LLMs return structured UI directly in conversation instead of plain text. Compatible with ShadCN, Material, and other design systems; supports OpenAI, Anthropic, Gemini, and other major models.

https://www.openui.com/

### More Resources

- **300 FUI Elements Pack** - A pack of 300+ futuristic UI elements for After Effects, editing, film, and motion design. https://nawazalamgir.gumroad.com/l/300fui
- **DesEngs** - A resource directory for design engineers, aggregating component libraries, case studies, and references. Content is still thin. https://desengs.com/

## Cases

### Natural

Payment infrastructure for AI agents. The design style nods to Cursor but maintains its own refinement.

https://www.natural.co/

### mem_brane

The standout piece this week. Through self-referential loops, ambient voice absorption, and transparent LED structures, it visualizes AI consciousness as a perceivable spatial experience. Truly not something you can vibe-code into existence — you have to watch the video.

https://www.nonfigurativ.com/projects/mem_brane

### Locale

The author behind the interfacecraft design tutorials vibe-coded a small tool — a web experiment that accesses local font permissions to display the machine's installed typeface families. More of an exploration of browser capabilities and typographic environments. A designer's taste really does show through.

https://joshpuckett.me/locale

### Matthew Petersen

A personal site I really like. The showcased projects are genuinely creative.

https://www.matthewpetersen.ca/
